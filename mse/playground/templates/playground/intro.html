Border Crossing Playground

Interactive geospatial analytics of U.S. border crossings powered by open government data, Plotly charts, and Leaflet maps — focused on the years 2020–2025.

Explore Map Port Analytics
Live fetch 2020–2025 Border · Measure · Value

Project Snapshot

  • Loads U.S. border data from the Transportation.gov open API.
  • Aggregates Value by port and year 2020–2025.
  • Maps ports with Leaflet, sized by total Value.
  • Uses Plotly for bar, line, and bubble charts per port and year.

Built as a standalone playground app inside your mse project — no impact on other apps.

What You Can Explore

Use the map for a high-level overview of border crossing intensity and the analytics view for per-port Value trends over time.

Encounters Map

A Leaflet map where each port is a circle marker whose radius represents total Value across 2020–2025. Popups show port name, border, state, and totals.

Port Analytics

For each selected port, see a Plotly bar + line chart of annual Value, and explore bubble charts showing total Value by port for a specific year.
